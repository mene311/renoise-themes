## 📢 Renoise Forums Post

**Title:** New web tool for browsing/creating Renoise color themes - preview in browser!

Hi everyone,

Just wanted to share a little project I've been working on: **Renoise Theme Share** (renoisethemes.com)

It's a web app that lets you:
- Browse community-made color themes (.xrnc files)
- See accurate previews of how they look in Renoise (Pattern Editor, Mixer & Waveform)
- Create your own themes using an interactive color palette generator
- Download themes with one click
- Like, comment, and follow other creators

The previews work by analyzing the actual .xrnc files and rendering them using pixel maps - so you see exactly how the theme would look in Renoise, without needing to install anything.

The site is **LIVE NOW** at renoisethemes.com with a coming soon page while we prepare for the full launch! Currently in final testing, but would love to get some feedback from the community! The site features both light and dark themes, tag-based filtering, and a creator tool with preset color schemes (monochrome, complementary, triadic, etc.)

If you're interested in taking a look, let me know and I can share access. Would be great to hear what you think!

Keep tracking! 🎵
