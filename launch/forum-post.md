Hi everyone,

I built a web platform for Renoise color themes and it's now live:

**https://renoisethemes.com**

**Right now you can:**
- Browse 8 handcrafted themes with pixel-accurate Pattern/Mixer/Waveform previews
- Use the **Theme Studio** — guided creation with archetypes, contrast auditing, design trap detection, and algorithmic Smart Fill
- Use the **Advanced Editor** — all 70+ Renoise UI elements, palette generation modes (monochrome, complementary, triadic), VU meter presets
- Upload, download, and share .xrnc files
- Auto-tagging by mood and color family

**How previews work:** a three-pass pixel recoloring pipeline maps every pixel in a reference screenshot to its UI element, then paints it with the theme's colors. The result is identical to opening the theme in Renoise.

Free, no ads. Feedback welcome via the form on the site.
